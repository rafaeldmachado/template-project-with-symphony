# References

This directory contains reference material that agents can consult when working
with specific technologies. Store `llms.txt` files and API references here.

## What to put here

- `llms.txt` files for libraries and frameworks you use
- API documentation excerpts for external services
- Design system references
- Deployment platform docs

## Naming convention

`<technology>-llms.txt` or `<technology>-reference.md`

## Example

If you're using a design system, put the reference here:
```
docs/references/design-system-reference-llms.txt
```

Agents will read these when working on related code, so keep them
focused and up to date.
